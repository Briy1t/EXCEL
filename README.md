# Mi Ruta de Aprendizaje de Excel
Autor: Briyit
Propósito: Construir bases sólidas en Excel para avanzar hacia un perfil analítico.

##  Descripción del repositorio
Este repositorio documenta mi proceso de aprendizaje de Excel profesional, combinando cursos, práctica autodidacta y ejercicios propios.
Mi objetivo es desarrollar una base sólida que me permita trabajar con datos, crear informes, automatizar tareas y, más adelante, avanzar hacia herramientas más potentes como Power Query, Power Pivot y eventualmente Power BI.

---
Aquí registraré:
- Apuntes y conceptos clave
- Ejercicios prácticos
- Archivos de práctica
- Mini‑proyectos
- Un proyecto final tipo dashboard
